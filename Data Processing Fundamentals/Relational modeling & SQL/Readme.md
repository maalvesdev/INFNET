# Assessment Final de SQL e Modelagem Relacional

Este repositório contém a resolução integral do Assessment da disciplina de SQL e Modelagem Relacional (Engenharia de Software - Instituto Infnet). O projeto foi desenvolvido inteiramente em um ambiente de notebook interativo (**Deepnote**), utilizando blocos nativos de SQL para demonstrar proficiência em consultas complexas e arquitetura de dados.

---

## Tecnologias e Ferramentas Utilizadas

* **Linguagem:** SQL
* **Ambiente de Desenvolvimento:** Deepnote (Notebooks interativos)
* **Conceitos Aplicados:** DDL, DML, JOINs, Agregações, Modelagem Relacional, Normalização (1FN, 2FN, 3FN).

---

## Estrutura do Assessment

O assessment está dividido em duas frentes principais de conhecimento estrutural:

### Parte A — Queries e Manipulação de Dados (12 Exercícios)
Foco em extração e transformação de dados sobre dois bancos distintos, aplicando:
* Implementação de comandos DDL para criação e estruturação inicial.
* Manipulação e atualização de registros (DML).
* Criação de consultas complexas utilizando múltiplos `JOINs` para cruzamento de tabelas.
* Uso de funções de agregação (`GROUP BY`, `COUNT`, etc.) para geração de relatórios de negócio.

### Parte B — Projeto de Modelagem (4 Exercícios)
Foco em arquitetura e design de software a partir de um cenário real de negócios (Marketplace):
* Análise de entidades e definição de relacionamentos.
* Aplicação prática de regras de normalização para eliminação de anomalias e redundâncias.
* Implementação física do *schema* projetado, traduzindo o modelo conceitual para código SQL real.
