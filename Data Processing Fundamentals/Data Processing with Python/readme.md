# Assessmento final de Python para Processamento de Dados

Este repositório contém a resolução do Assessment da disciplina de Python para Processamento de Dados (Engenharia de Software - Instituto Infnet). O projeto demonstra desde o domínio de estruturas de dados nativas e lógica de programação até o desenvolvimento de serviços web e consumo de APIs externas de Inteligência Artificial.

---

## Tecnologias e Ferramentas Utilizadas

* **Linguagem:** Python
* **Framework Web:** Flask
* **Bibliotecas:** `requests`, `json`
* **Integrações:** API OpenRouter (Modelos de IA)
* **Conceitos Aplicados:** Estruturas de Dados, Manipulação de Arquivos, Tratamento de Exceções, Protocolo HTTP, REST APIs.

---

## Estrutura do Projeto

O assessment foi dividido em duas fases de complexidade progressiva:

### Parte A — Lógica de Programação e Estruturas de Dados (Exercícios 1 a 12)
Foco na resolução de problemas algorítmicos utilizando **Python puro**:
* Manipulação e fatiamento avançado de **Strings**.
* Uso eficiente de coleções de dados: **Listas, Dicionários e Sets**.
* Leitura, escrita e processamento de **Arquivos e JSON**.
* Implementação de rotinas de **Tratamento de Erros e Exceções** (`try/except`), garantindo a robustez dos scripts.

### Parte B — Desenvolvimento Web e IA (Exercícios 13 a 16)
Foco em arquitetura Back-End e comunicação entre sistemas:
* Construção de um pequeno serviço web utilizando o microframework **Flask**.
* Criação de *endpoints* para processamento de requisições.
* Implementação de requisições HTTP utilizando a biblioteca `requests`.
* **Integração com IA:** Autenticação e consumo da API do **OpenRouter**, extraindo e processando respostas no formato JSON com aplicação de conceitos de Engenharia de Prompt.
