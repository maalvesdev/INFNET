# Assessment Final de Introdução a Programação com Python

Este repositório contém a resolução integral do Assessment da disciplina de Introdução a Programação com Python (Engenharia de Software - Instituto Infnet). O foco deste projeto é demonstrar o domínio dos fundamentos da linguagem, aplicados à resolução de problemas complexos de negócio, cálculos financeiros, validação de dados e modelagem matemática.

---

## Tecnologias e Conceitos Aplicados

* **Linguagem:** Python
* **Bibliotecas Embutidas:** `math`, `random`
* **Ambiente de Desenvolvimento:** Deepnote / Jupyter Notebooks
* **Estruturas de Dados:** Manipulação avançada de Listas e Strings.
* **Lógica e Controle:** Laços de repetição (`while`, `for`), controle de fluxo e funções modulares.
* **Algoritmos Aplicados:** Validação de checksum (CPF), Método de Monte Carlo (Probabilidade), e conceitos fundamentais de Cálculo (Limites, Derivadas e Integrais aproximadas via código).

---

## Estrutura do Projeto

Os 16 exercícios foram desenvolvidos para resolver cenários práticos e estão divididos nas seguintes áreas de conhecimento:

### 1. Processamento de Dados e Regras de Negócio
* **Cálculo de Custos e Precificação:** Algoritmos para cálculo unitário e de lotes de manufatura de precisão.
* **Formatação e Triagem de Dados:** Padronização de strings e concatenação de dados cadastrais de pacientes.
* **Análise Financeira:** Simulação de crescimento de capital usando juros compostos, com regras dinâmicas de bônus e penalidades.
* **Gestão de Metas:** Loop de monitoramento em tempo real de doações comunitárias usando somatórios acumulados.

### 2. Algoritmos de Validação e Comparação
* **Validação de CPF:** Implementação do algoritmo oficial da Receita Federal utilizando cálculos de dígitos verificadores e pesos matemáticos.
* **Auditoria de Estoques:** Funções para cruzar listas de dados e encontrar interseções precisas entre armazéns.
* **Sistema de Avaliação:** Comparação iterativa de listas (gabaritos oficiais vs. respostas) para ranqueamento de candidatos.

### 3. Modelagem Matemática e Estatística
* **Método de Monte Carlo:** Simulação probabilística utilizando milhares de pontos aleatórios bidimensionais para estimar o valor de Pi.
* **Aproximações Numéricas (Cálculo):** * Cálculo de limites investigando a convergência de funções por aproximação.
  * Cálculo de derivadas (velocidade instantânea) a partir do fatiamento de tempo em arrays de posição.
  * Cálculo de integrais (área sob a curva) pelo método de soma de retângulos para otimização de compra de materiais.

### 4. Sistemas Interativos e Matrizes
* **Motor de Movimentação em Grid (2D):** Desenvolvimento de um protótipo de jogo em texto rodando em loop contínuo.
* **Gerenciamento de Estado e Colisões:** Validação de coordenadas (X, Y) em uma matriz 20x20, renderização do mapa no console e detecção de barreiras e objetivos.
